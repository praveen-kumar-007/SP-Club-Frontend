# SP Club Website Frontend

## Overview
This is the frontend repository for the SP Club Website project. It's built using modern web technologies to provide a seamless user experience for club members and administrators.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation
1. Clone the repository
```bash
git clone https://github.com/your-username/SP-Club-Frontend.git
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

## Features
- User authentication
- Club event management
- Member registration
- Activity tracking
- Administrative dashboard

## Tech Stack
- React.js
- Next.js
- Tailwind CSS
- TypeScript
